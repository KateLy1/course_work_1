# course_work_1
Здесь хранятся файлы с кодом, с помощю которого была реализована курсовая работа 1 курса магистратуры ФКН программы АДД.

В папке отзывы_таблицы находятся отзывы клиентов о недвижимости, собранные с сайта Яндекс.Недвижимость в файле Книга3.xlsx. В папке также есть код для обработки yandex2.ipynb и результат обработки отзывов reviews2.csv.

В папке общий_парсинг находится файл с кодом KP.ipynb и данные, получаемые по ходу cian_parsed.csv содержит данные парсинга основных страниц Циана (где представлены разные квартиры в новостройках), в зип архиве карточки.zip находятся файлы с кодом страниц новостроек (на каждой странице информация только об одной новостройке), также в папке есть 5 csv файлов, полученных постепенной обработкой отдельных страниц квартир с сайта и файл buildings.csv итоговый файл.

В папке кластеризаиция проводится кластеризация по отзывам clean_reviews.ipynb входной файл reviews2.csv и по характеристикам новостроек clean_buildings.ipynb входной файл buildings.csv.

В папке классификация_новостроек находятся два способа классификации новостроек: через api гигачата classes_buildings.ipynb (результат buildings_classes.csv) и через определенные условия classes_buildings.ipynb(cond) (результат buildings_classified_optimal_value2.csv).

В папке лин_модели в файле lin_models.ipynb сожержится код с выявлением значимости атрибутов для цены и отрисовкой карт ценнсотей.
